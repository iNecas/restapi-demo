Tweets / index

GET /users/:user_id/tweets
List tweets

Examples

GET /users/980190962/tweets
200
[
  {
    "created_at": "2012-07-14T20:56:15Z",
    "id": 980190962,
    "scheduled_at": "2012-06-27T17:55:59Z",
    "sent": false,
    "text": "MyText",
    "updated_at": "2012-07-14T20:56:15Z",
    "user_id": 980190962
  }
]

Tweets / create

POST /users/:user_id/tweets
Create a tweet

Examples

POST /users/980190962/tweets
{
  "tweet": {
    "scheduled_at": "2012-06-27 17:55:59 UTC",
    "sent": false,
    "text": "MyText",
    "user_id": "980190962"
  }
}
201
{
  "created_at": "2012-07-14T20:56:15Z",
  "id": 980190963,
  "scheduled_at": "2012-06-27T17:55:59Z",
  "sent": false,
  "text": "MyText",
  "updated_at": "2012-07-14T20:56:15Z",
  "user_id": 980190962
}

Params

  • tweet : optional [ Has to be hash. ]
    • scheduled_at : optional
    • sent : optional [ Has to be a boolean ]
    • text : optional
    • user_id : optional [ Has to be a number. ]
