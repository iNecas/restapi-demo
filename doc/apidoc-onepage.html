POST /users/:user_id/tweets
Create a tweet

Examples

POST /users/980190962/tweets
{
  "tweet": {
    "scheduled_at": "2012-06-27 17:55:59 UTC",
    "sent": false,
    "text": "MyText",
    "user_id": "980190962"
  }
}
201
{
  "created_at": "2012-07-14T20:56:15Z",
  "id": 980190963,
  "scheduled_at": "2012-06-27T17:55:59Z",
  "sent": false,
  "text": "MyText",
  "updated_at": "2012-07-14T20:56:15Z",
  "user_id": 980190962
}

Params

Param name Description
tweet
optional
Value: Has to be hash.

optional
tweet[sent]
optional
Value: Has to be a boolean
tweet[text]
optional
tweet[user_id]
optional
Value: Has to be a number.
